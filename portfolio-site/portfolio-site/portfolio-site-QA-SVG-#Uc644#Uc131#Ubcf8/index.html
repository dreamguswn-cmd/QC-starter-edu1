Project 02 · Evaluation

Two-Stage Evaluation 기반
RAG 챗봇 자동 품질 평가

RAG 챗봇의 검색·생성 결과를 사람마다 다른 감각으로 평가하지 않고, 명확한 기준과 감점 규칙으로 자동 검증하는 품질평가 시스템입니다.

문제 정의

수작업 평가의 주관성과 반복 비용

평가자마다 점수가 흔들리고 테스트가 늘어날수록 검토 시간이 커지는 문제를 해결하고자 했습니다.

핵심 설계

루브릭과 감점을 분리

이해도·정확성·관련성·표현성은 루브릭으로 평가하고, 출처 미기재·근거 미사용은 별도 감점했습니다.

자동화 결과

진단 가능한 품질 리포트

점수뿐 아니라 감점 이유와 개선 코멘트를 Markdown 보고서로 자동 생성해 결과 추적성을 높였습니다.

  • Judge Agent와 Two-Stage Evaluation 평가 흐름 설계
  • 이해도·정확성·관련성·표현성 루브릭 기준 정의
  • 출처 미기재·정의만 제공·근거 미사용 감점 규칙 구현
  • 점수 인플레이션 사례 분석 및 루브릭 기준 보정
  • 항목별 점수·진단·개선 코멘트가 포함된 Markdown 리포트 자동화
