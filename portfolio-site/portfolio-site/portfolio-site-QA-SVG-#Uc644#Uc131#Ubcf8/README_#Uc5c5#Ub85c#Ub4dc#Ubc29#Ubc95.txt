GitHub Pages 업로드 방법

1. 이 ZIP 파일을 컴퓨터에서 압축 해제합니다.
2. GitHub 저장소의 portfolio-site 폴더로 이동합니다.
3. 압축을 푼 안쪽 파일과 폴더를 모두 업로드합니다.
4. index.html, styles.css, script.js, assets, downloads, portfolio.pdf가 같은 위치에 있어야 합니다.
5. Commit changes를 누르고 1~5분 기다린 뒤 Ctrl+F5로 새로고침합니다.
